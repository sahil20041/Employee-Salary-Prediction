# Employee Salary Prediction

A Machine Learning project that predicts employee salaries using demographic and professional attributes such as age, education level, gender, job category, and years of experience.
* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-Learn
* 
 Workflow
* Data Cleaning & EDA
* Feature Engineering
* Ordinal & One-Hot Encoding
* Standard Scaling
* Column Transformer & Pipeline
* Linear Regression
* Cross Validation
* 
Evaluation Metrics
* R² Score
* MAE
* MSE
* RMSE
